What is..

frontend?

Everything you see and experience in a webpage, application, mobile application or a system is called frontend.
It includes everything that you as a user experience directly: text, colors and styles, images, graphs and tables, buttons and navigation menu.
-Frontend is the interface with which the user interacts-
It prescribes how functions and information are to be presented, as well as how the website is to function in general, where focus is on usability and creating a good experience for the user. Check out the links in the round pictures to the right for more info!
